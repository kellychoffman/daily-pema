Daily Pema

A free iPhone widget. One Pema Chödrön quote every day, right on your home screen.

“You are the sky. Everything else — it’s just the weather.”

— Pema Chödrön

1 Install Scriptable
It’s a free app from the App Store that runs the widget.

Get Scriptable
2 Copy the script
Tap the button below to copy the script code to your clipboard.

If the button didn't work, tap inside the box, choose Select All, then Copy:

3 Paste it into Scriptable
Open the Scriptable app and tap the + button to create a new script. Paste the code into the editor. Tap the script's title at the top and rename it to DailyPema. Tap Done to save.
4 Add the widget to your home screen
Long-press an empty spot on your home screen. Tap Edit in the top-left, then tap Add Widget. Search for Scriptable, pick a widget size, and tap Add Widget. Then long-press the new widget, choose Edit Widget, and set the script to DailyPema.
